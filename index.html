Публичный API MEGAVISOR

Точка входа http://megavisor.com/export

Все методы могут возвращать данные в нескольких форматах: http://megavisor.com/export/*method_name*.*format* method_name — имя метода format - json, xml, csv, yaml, xls Например http://megavisor.com/export/*catalog*.*json* вернет каталог объектов в формате json

Доступные методы

Получение списка объектов

GET /catalog.json

Возвращает каталог Параметры

  • uuid - uuid каталога уникальный идентификатор коллекции
  • type=sequence|animation|panorama_sequence|panorama|video timestamp=asc|desc desc rating=asc|desc desc name=asc|desc asc editor=asc|desc desc tags=129,156 search=Some Search Text page=1 - страница для экспорта с 1
